## 1. Abstraction
Abstraction adalah konsep dalam OOP yang menyembunyikan detail implementasi dan hanya menampilkan fungsi atau fitur yang penting kepada pengguna.

## 2. Encapsulation
Encapsulation adalah konsep menyatukan data (atribut) dan method (fungsi) dalam satu class, serta membatasi akses langsung ke data tersebut.

## 3. Inheritance
Inheritance adalah konsep dalam OOP di mana sebuah class dapat mewarisi atribut dan method dari class lain.

## 4. Polymorphism
Polymorphism adalah konsep dalam OOP di mana satu method atau objek dapat memiliki banyak bentuk atau perilaku yang berbeda.
